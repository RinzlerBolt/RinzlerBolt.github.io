Project Overview

Development Time: Jan 2024 - March 2024

Engine: Unity

Genre: Simulator

Team Size: 1

My Role: Sole Developer

Software Used:

My Contributions

  • Implemented Basic AI Movement Systems.
  • Developed Pathfinding, Jump Point Search, A*, Djikstra's Algorithms
  • Made a User Interface on the Interactions the AI had to the Game
  • Created an AI Behaviour Tree System, with Nodes and Queries. Like UE5's Behaviour Tree

Project Details

We were given 12 AI tasks to work with, and create AI behaviors for each of them, they ramped up in difficulty over time. From simply following the player character, to attacking, to creating AI pathfinding algorithms like Jump Point Search, A*, and Djikstra's. It was a challenge for sure, and allowed me to express my creativity in creating a very smooth flowing AI. My favourite AI development was A*.
