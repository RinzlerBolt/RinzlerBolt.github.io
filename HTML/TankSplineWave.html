My Contributions

  • Developed a Mesh Generation Tool using Splines, then Simulated a road and a car Driving down it.
  • Developed on a Realsitic Tank, with a slow moving turret and firing (Dirty Flag), Tank Movement, and a fully working suspension
  • Developed An Advanced Wave Spawner with ECS, can be modified down to the Enemy Data and Wave Spawning Behaviour

Project Details

This project was a task set by Staffordshire University, where we were challenged to develop three games featuring Spline Generation, a Tank Controller, and a Wave Spawner. The complexity lay in integrating everything we had learned while writing efficient, modular scripts. I made it a personal goal as a developer to avoid relying on the Update methods unless absolutely necessary. Each system was designed to be highly modular, allowing for easy modification by designers. The Wave Spawner, in particular, was designed to be extremely flexible, with its behavior easily adjustable through simple variable changes.
